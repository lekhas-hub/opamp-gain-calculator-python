# Op-Amp Gain Calculator (Python)

## 📌 Description
This project calculates the gain of inverting and non-inverting op-amp circuits.

## 🚀 Features
- Inverting amplifier gain
- Non-inverting amplifier gain
- Simple user input

## 🧠 Concepts Used
- Python
- Analog Electronics
- Operational Amplifiers

## ▶️ How to Run
Run:
python main.py

## 💡 Example
Inverting:
Gain = -Rf / Rin

Non-Inverting:
Gain = 1 + (Rf / Rin)
